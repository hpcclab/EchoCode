# UserImplementation for Annotations_BigO

Set `echocode.featureImplementation.annotationsBigO` to `user` to enable overrides.

Supported override files in this folder:

- `bigOAnalysis.js` exports `registerBigOCommand`
- `annotations.js` exports `registerAnnotationCommands`

If a file is missing or exports are invalid, EchoCode falls back to built-in behavior.
