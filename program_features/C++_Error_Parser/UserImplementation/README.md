# UserImplementation for C++\_Error_Parser

Set `echocode.featureImplementation.errorParser` to `user` to enable overrides.

Supported override files in this folder:

- `CPP_Error_Parser.js` exports `compileCurrentCppFile`
- `Python_Error_Parser.js` exports `checkCurrentPythonFile`

If a file is missing or exports are invalid, EchoCode falls back to built-in behavior.
